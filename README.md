# ISL Wireless Private Limited Website

Production-style multi-page React + Vite website for ISL Wireless Private Limited.

## Pages
- Home
- About
- Solutions
- Industries
- Projects
- Contact

## Run locally
```bash
npm install
npm run dev
```

## Build for production
```bash
npm install
npm run build
```

## Notes
This package uses placeholder contact details and placeholder branding text where official assets/details were not provided.
Replace these before publishing:
- logo and favicon

- official email and enquiry form endpoint
- project photographs / case studies
